# an0306 · Yue System Workspace

這個 repo 是 An An 機器上「玥」(Yue) 的工作空間快照，用來做備份與版本管理。

## 結構總覽

- `SOUL.md` / `IDENTITY.md`：
  - 定義玥的人格、風格與核心原則。
- `AGENTS.md` / `AUTO-DISPATCH.md` / `SYSTEM_ARCHITECTURE.md`：
  - 描述整個系統的代理架構、分工與任務流。
- `USER.md`：
  - 關於使用者（An An）的基本資訊與偏好。
- `MEMORY.md`：
  - 長期核心記憶（Golden Memory）。
- `memory/`：
  - 每日記錄、索引、痛點分析與檢索審計等。
- `scripts/`：
  - 與記憶編碼、檢索與衰減相關的工具腳本。
- `subagents/`：
  - 子代理（例如分析、工匠、監控）相關設定與說明。
- `Sacred-Essence-Viking/`：
  - 外部記憶系統引擎（作為子模組/外部 repo），用於更進階的記憶矩陣實驗。

## 用途

- 作為這台機器上 Yue 系統的「真實狀態備份」。
- 方便比對不同時間點的設定、記憶與腳本變化。
- 未來可以作為其他機器/實例部署類似系統的參考範本。

> 注意：這個 repo 主要對應 `/root/.openclaw/workspace`，實際執行行為仍由本機的 OpenClaw Gateway 決定。
